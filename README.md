 🚀 Guardian Cloud

An AI-powered cloud-based content moderation system built using AWS services.

 🔧 Tech Stack

* AWS (Lambda, API Gateway, DynamoDB, Cognito)
* JavaScript (Frontend)
* HTML, CSS (UI)
* AI-based moderation logic

 ✨ Features

* 🔐 User authentication with AWS Cognito
* 💬 Real-time comment moderation
* 🚫 Toxic content detection & blocking
* 📊 Live dashboard for threat monitoring
* ⚡ Serverless architecture using AWS Lambda

 📡 How It Works

1. User submits a comment
2. Comment is sent to AWS API Gateway
3. Lambda processes and classifies content
4. Response updates UI in real-time

 ⚠️ Note

API endpoints are not included for security reasons.
Replace API_URL with your own AWS deployment.
